Core Features

Authentication Service

Secure user management with JWT authentication, password hashing, and user profile management. Implement login, registration, and token refresh with minimal code.

Real-time Database

JSON-like hierarchical data structure with WebSocket-based real-time updates. Get instant data synchronization across clients with path-based data access and optimized caching.

Security Rules Engine

Granular access control with path-based permission structure and rule inheritance for hierarchical data. Define who can read and write to your database with a developer-friendly rule syntax.

Client SDK

Firebase-compatible API with real-time data synchronization, authentication methods, and easy integration with web applications. Build apps faster with our intuitive client library.

AI-Powered Database Operations

Harness the power of generative AI to create, manage, and interact with your database effortlessly. The Instantiate.dev AI assistant makes database operations accessible through natural language.

  • ✓ Create database structures from natural language descriptions
  • ✓ Query your data using plain English questions
  • ✓ Update your data by describing the changes you want
  • ✓ Analyze your database structure for best practices
  • ✓ Chat with an AI assistant for database help and guidance
// Create a blog database with AI
const response = await fetch('/api/ai/create-database', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    prompt: 'I need a database for a blog with users, posts, and comments'
  })
});

const data = await response.json();
console.log(data.structure); // The generated database structure
// Query data with natural language
const response = await fetch('/api/ai/generate-query', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    prompt: 'Show me all posts by user1 with their comments'
  })
});

const data = await response.json();
console.log(data.query); // The generated query code

Getting Started

Docker Deployment

# Generate security keys
npm run generate-keys

# Launch with Docker Compose
docker-compose up -d

Manual Deployment

# Install dependencies
npm install

# Configure environment
cp .env.example .env
npm run generate-keys

# Start the server
npm start
